# Quagga / Квагга
## Инструмент для обучения, интегрируемый примерно везде :)
#### Запуск:

```
npm install
```

1. Сайт:

```
npm start
```

2. Декстопное приложение:

```
npm run app
```

3. Сервер:

```
python app.py
```

Продолжение: 

Тг-бот: https://github.com/Anisa6/quagga-bot
```
python qqq.py
```
